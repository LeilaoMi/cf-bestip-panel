# 版本说明

## 当前能力

项目已完成 Cloudflare Pages 适配、数据源管理、数据产物构建、配置安全、测速闭环、UI 改造和完整测试体系。

## 关键变更

### Cloudflare Pages 适配

- 增加 Pages Functions API。
- 增加 `_headers` 安全头和缓存策略。
- 线上禁用动态构建接口。
- 支持 GitHub Actions 定时构建数据产物。

### 数据产物

- 支持按类型拆分输出。
- 支持 `full.json`、`health.json` 和 `manifest.json`。
- 支持本地目录和 public 静态目录同步。

### 配置管理

- 增加导入 dry-run 校验。
- 增加页面内导入预览。
- 增加保存前差异摘要。
- 增加保存快照、本地归档、搜索筛选和恢复预览。

### 测速模块

- 修复浏览器裸 IP HTTPS 测速不可靠问题。
- 本地支持 TCP connect 测速。
- 线上支持候选导出、cfst 命令导出、CSV 导入。
- 增加质量筛选、结果导出和历史稳定性对比。

### UI

- 调整为 Cloudflare SaaS + 企业后台混合风。
- 移除外部字体依赖，改用系统字体栈。
- 保留清晰的仪表盘、表格、按钮和筛选控件。

### 测试体系

- 增加单元测试。
- 增加 UI Smoke。
- 增加统一测试报告。
- 增加部署前检查。

## 后续建议

- 接入更完整的线上监控。
- 增加 Cloudflare Pages 部署后的自动验收脚本。
- 为大型数据源增加更细粒度的失败原因统计。
- 根据实际使用情况优化测速历史展示。
