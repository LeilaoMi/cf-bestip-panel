## 注意事项

- 暂不线上部署；部署、GitHub push、域名绑定前必须再次确认。
- `/api/build-dataset` 使用 Node 子进程和本地文件写入，只适合 Zo/Node 本地服务；Cloudflare Pages 线上应改为 GitHub Actions 或 Cron 产物构建。
- 不要写入任何 token、cookie、私有订阅密钥。
